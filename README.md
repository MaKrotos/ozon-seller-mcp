# ozon-seller-mcp 🦠

MCP-сервер для [Ozon Seller API](https://docs.ozon.ru/api/seller/) — позволяет AI-ассистентам (Claude, OpenClaw и др.) полноценно управлять вашим магазином на Ozon.

## 🌟 Возможности

- **📦 Управление товарами**
    - Просмотр, создание, обновление, архивация и удаление.
    - **Новое:** Получение детальных характеристик через `/v4/product/info/attributes`.
    - **Новое:** Полный импорт и редактирование через `/v3/product/import`.
    - **Новое:** Отслеживание статуса асинхронного импорта по `task_id`.
    - Загрузка изображений.
- **💰 Цены и Остатки**
    - Мониторинг и обновление цен.
    - Управление остатками (FBS/FBO) в реальном времени.
- **🚚 Заказы (FBS/FBO)**
    - Полный цикл обработки FBS заказов: от списка до печати этикеток и трекинга.
    - Мониторинг FBO отгрузок.
- **📂 Категории и Склады**
    - Работа с деревом категорий и справочниками атрибутов.
    - Управление складами продавца.
- **📈 Аналитика и Финансы**
    - Сбор данных по продажам, просмотрам и конверсиям.
    - Финансовые транзакции, итоги и отчеты по реализации.
- **💬 Коммуникации**
    - Интеграция с чатами покупателей (чтение/отправка).
    - Работа с вопросами и отзывами.
- **⭐ Рейтинг и Акции**
    - Мониторинг рейтинга продавца.
    - Управление участием товаров в акциях Ozon.

## 🛠 Установка

```bash
# Клонировать репозиторий
git clone https://github.com/MaKrotos/ozon-seller-mcp.git
cd ozon-seller-mcp

# Установить зависимости через uv
uv sync
```

## ⚙️ Настройка

Получите `Client ID` и `API Key` в личном кабинете Ozon Seller $\to$ Настройки $\to$ Seller API.

Создайте файл `.env` в корне проекта:
```bash
OZON_CLIENT_ID=your_client_id
OZON_API_KEY=your_api_key
```

## 🤖 Интеграция с AI-клиентами

### Claude Desktop / OpenClaw
Добавьте в конфигурационный файл:

```json
{
  "mcpServers": {
    "ozon-seller": {
      "command": "uv",
      "args": ["run", "--directory", "/path/to/ozon-seller-mcp", "python", "server.py"],
      "env": {
        "OZON_CLIENT_ID": "your_client_id",
        "OZON_API_KEY": "your_api_key"
      }
    }
  }
}
```
